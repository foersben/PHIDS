Substance Configuration Guide

Columns Explained:

  • 🆔 ID: The internal index (0 to 15) corresponding to the substance's dedicated layer index in signal/toxin grids.
  • 🏷️ Name: Human-readable label for custom identification.
  • 🧪 Type: Behavioral behavior of the chemical:
    • Signal: Volatile airborne chemical that diffuses to warn neighboring plants.
    • Lethal Toxin: Local poison that kills grazing herbivores.
    • Repellent Toxin: Local irritant that forces herbivores to move away.
    • Toxin: Standard chemical deterrent affecting feeding efficiency.
  • ⏱️ Synth. Dur.: Number of ticks the plant continues synthesizing the substance once triggered.
  • ⏳ Aftereffect: Remaining ticks the substance persists in plant cells after synthesis stops.
  • ☠️ Lethality: Rate of population reduction per tick applied to swarms feeding on plants producing lethal toxins.
  • 👥 Min Pop.: The minimum herbivore headcount required in the cell to trigger the substance's production.
  • 🛠️ Actions: Add new substances (up to 16) or delete existing ones.

How to Apply Changes?

Modifying any value in a table field immediately triggers a background save to the server-side draft.

⚠️ Crucial: These edits will not affect the active running simulation loop until you click the 🔄 Load Draft button in the sticky top header, or restart/reset the simulation loop.
