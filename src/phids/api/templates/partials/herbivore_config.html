Field interpretation for herbivore species

ID is the species index used by diet and trigger matrices. Name is the display label shown in legends, tooltips, and dashboard summaries.

Emin defines the per-individual baseline reserve used by upkeep, starvation attrition, and reproduction thresholds. Speed is ticks between moves (1 means movement every tick).

Eat Rate controls plant-energy extraction while feeding. Repr. ÷ controls the energy surplus required for fission-style growth (higher values delay offspring generation).

Actions removes the species and compacts dependent references. Diet compatibility is configured in the 🍽️ Diet Matrix, where each herbivore row maps edible flora columns.
