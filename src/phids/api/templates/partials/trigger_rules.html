⚡ Trigger Rules

Each rule defines: when a plant species is attacked by a herbivore with enough population, it synthesises a substance. Multiple rules per (plant, herbivore) pair are allowed — they fire simultaneously. Advanced activation gates are configured visually as nested trees with all_of / any_of groups over enemy_presence and substance_active predicates.

How it works: Each tick, the engine checks every plant's cell for herbivores. If a matching herbivore swarm meets the minimum population, synthesis begins. Synthesis takes N ticks (from the Substance config). Once synthesised, the optional activation gate is evaluated. Gates can mix enemy and substance predicates visually through nested all_of/any_of groups. Multiple rules on the same (plant, herbivore) pair fire simultaneously and independently.
