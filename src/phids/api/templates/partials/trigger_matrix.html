Substance Trigger Matrix Guide

Matrix Layout & Interactions:

This grid couples plant defense with specific herbivore attacks.

  • Rows: The active flora/plant species in the simulation draft.
  • Columns: The active herbivore species in the simulation draft.
  • Dropdown Selection: Choose which substance is synthesized when the corresponding herbivore attacks the plant species.
  • — none —: Disables automatic substance synthesis for this (flora, herbivore) pair.

How to Apply Changes?

Changing any dropdown selection immediately triggers a background save to the server-side draft.

⚠️ Crucial: These edits will not affect the active running simulation loop until you click the 🔄 Load Draft button in the sticky top header, or restart/reset the simulation loop.
