Biotope Configuration Guide

Core Biotope Fields Explained:

  • 📐 Grid Dimensions: Width & Height of the cellular automaton board (10x10 to 80x80). Changing these clears placements.
  • ⏱️ Max Ticks & Speed: Controls simulation termination limits and default tick rate in Hz.
  • 💨 Wind Vector: The directional vector (X, Y) determining the drift/advection direction of volatile signals and toxins across cells.
  • 🔬 Signal/Toxin Layers: Number of available slot layers for chemical signals or toxins (1 to 16).
  • 🤝 Mycorrhizal Inter-Species: If enabled, allows mycorrhizal networks (root links) to form between different plant species, enabling mutual resource sharing.

How to Apply Changes?

Edits to inputs auto-save to the backend draft immediately.

⚠️ Crucial: To update the live running simulation, you must click 🔄 Load Draft, or start/reset the simulation loop.
