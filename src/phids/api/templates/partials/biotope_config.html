Biotope Configuration Guide

Core Fields Explained:

  • 📐 Grid Dimensions: Width & Height of the cellular automaton board (10×10 to 80×80). Larger grids allow more spatial heterogeneity but increase computation.
  • ⏱️ Max Ticks & Speed: Controls simulation termination limits and default tick rate in Hz.
  • 💨 Wind Vector (X, Y): Drives semi-Lagrangian advection of airborne signal grids. Positive X shifts signals east; positive Y shifts them south. Wind does not affect mycorrhizal relay.
  • 🔬 Signal/Toxin Layers: Number of available slot layers for chemical signals or toxins (1–16). Governed by the Rule-of-16 for L1/L2 cache efficiency.
  • 🤝 Mycorrhizal Inter-Species: If enabled, root networks form between different plant species, enabling cross-species chemical alarm sharing.

Diffusion & Emission Parameters:

  • 🌫️ Signal Decay Factor (0.01–1.0): The fraction of signal concentration retained per tick after Gaussian diffusion. Default 0.85 means 15% of airborne concentration dissipates each tick. Set to 1.0 for persistent signals; set to 0.3 for rapidly fading alerts. This directly controls how far chemical alarms can propagate spatially before becoming undetectable.
  • 📡 Substance Emit Rate (0.01–1.0): The concentration increment added to a signal or toxin layer per tick when an active substance emits. Higher values cause layers to saturate faster. For mycorrhizal relay, the total emit is split: airborne = rate / (relay_targets + 1); the remainder is distributed across root-connected neighbours.

Z-Rules (Termination Circuit-Breakers):

  • Z2 — Flora Extinction: Halts the simulation when a specific flora species index goes completely extinct. Set to -1 to disable. Useful for testing whether a herbivore species can drive a flora species to zero.
  • Z4 — Herbivore Extinction: Halts when a specific herbivore species goes extinct. Useful for measuring the effectiveness of a chemical defense strategy.
  • Z6 — Total Flora Energy Cap: Halts if the total energy summed across all plants exceeds this value. Acts as an ecological carrying-capacity limiter to prevent unchecked plant growth.
  • Z7 — Herbivore Population Cap: Halts if the total herbivore headcount across all swarms exceeds this value. Prevents runaway population explosions that would crash the simulation.
