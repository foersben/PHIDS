❓ Guide

Reference for interpreting every live-grid overlay, tooltip section, diagnostic number, and how the displayed values are computed.

How is each cell rendered?
Cell fill color is the current flora energy layer for each species; brighter fill means more stored plant energy in that grid cell. Colored dots are herbivore swarms grouped by species. Blue lines are active mycorrhizal links between adjacent plants. Blue rings indicate plants with visible signal activity (active, synthesizing, or lingering signal substances), and purple squares indicate analogous toxin activity. Cyan/purple screen overlays show the current per-cell maximum signal/toxin concentration across all configured substance layers.
How should I read the top of the tooltip?
The header always identifies one grid cell only. The tick badge is the current simulation step of the live loop. flow is the current flow-field value read from env.flow_field[x, y]. wind is the uniform wind vector sampled from the environment wind matrices at that same cell. signal peak is max(signal_layers[:, x, y]); toxin peak is max(toxin_layers[:, x, y]). The “Signals here” and “Toxins here” lists then break those peaks back into per-substance concentrations for the hovered cell.
What do the plant properties mean?
For each plant, energy / max energy is its current reserve relative to the species cap. growth is the configured growth-rate parameter, camouflage reports whether the flow-field attenuation is enabled, and root links is the number of current mycorrhizal neighbors. Each substance card shows whether it is active or just triggered, whether it is lethal or repellent, how many synthesis/aftereffect ticks remain, and the gate line summarizing the activation predicate tree.
What do swarms and mycorrhiza properties mean?
Swarm population is current headcount, baseline is the spawn population used for growth/mitosis comparisons, signal and toxin are the local overlay concentrations at the hovered cell, and status lines report whether the swarm is energy-stressed, repelled, or intoxicated. The Mycorrhiza block counts all root links touching the hovered cell, lists whether inter-species links are allowed, shows connection cost and signal velocity settings, and enumerates each linked neighbour cell.
What does the diagnostics sidebar show?
Model shows draft counts, live entity counts, the newest telemetry record, and an energy-deficit watch driven by current swarm reserves. Frontend reports the dashboard controller state and WebSocket/render status. Backend shows recent in-memory PHIDS logs from API requests, draft mutations, engine transitions, telemetry, and websocket activity.
