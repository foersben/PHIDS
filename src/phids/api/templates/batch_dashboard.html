🧬 Monte Carlo Batch Runner

Execute N independent simulation trajectories in parallel using a ProcessPoolExecutor (spawn context). Each run is seeded independently to generate ensemble diversity while preserving per-seed determinism. Aggregate statistics (mean ± σ) are computed across the ensemble and persisted for export.
